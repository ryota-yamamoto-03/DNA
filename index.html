山本涼太（Ryota Yamamoto）— Creative Developer ポートフォリオ

山本涼太は Web・Game・Video・AI の分野で活動するクリエイティブデベロッパーです。

制作実績

WowMe — Web

推しとファンをつなぐ動画メッセージアプリ。特別な想いを届ける新体験サービスで誕生日や記念日に世界に一つの感動を贈れるサービスです。

使用技術: HTML, CSS, JS, Square

STOPLINE — Game

制限時間内に線を引いてボールを目的地まで運ぶ、戦略性と直感が試されるシンプルなシングルプレイゲームです。

使用技術: Unity, C#, WebGL

第六感 歌詞動画 — Video

第六感 / Reolの歌詞動画。After Effectsで、音楽のビートや歌詞の内容に合わせてダイナミックに変化するアニメーションを実現。

使用技術: After Effects

ドルスケ — AI

アイドルのライブ情報をまとめて確認できるスケジュール共有サービス。推し活を効率化できる便利ツールです。

使用技術: ClaudeCode, Supabase, Vercel

MetaMuse — Web

仮想空間で活動する次世代バーチャルアイドル。音楽とテクノロジーを融合し新しい推し体験を届ける存在です。

使用技術: ChatGPT, suno AI, HTML, CSS, JavaScript

これがぼくのきもち — Video

スカイピースさんのMVをUnityで完全再現。UnityのTimelineを駆使して作成したファンメイドの映像作品です。

使用技術: Unity

AI Closet — AI

AIがあなたのクローゼットから今日の天気に合ったコーデを提案します。

使用技術: ClaudeCode, Supabase, Vercel
